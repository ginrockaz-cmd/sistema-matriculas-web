# Sistema de Gestión de Matrículas

## 📌 Descripción
Sistema web desarrollado para automatizar el proceso de matrícula en una institución educativa, optimizando el registro de estudiantes, control de pagos y generación de reportes.

## 🚀 Funcionalidades
- Registro y gestión de estudiantes
- Gestión de matrículas
- Control de pagos
- Generación de estados de cuenta
- Exportación de reportes en PDF

## 🛠 Tecnologías utilizadas
- PHP
- MySQL
- HTML
- CSS
- JavaScript
- FPDF (generación de PDF)
  
## ⚙️ Instalación y uso

1. Clonar el repositorio
2. Importar la base de datos (archivo .sql) en MySQL
3. Configurar la conexión en config.php
4. Ejecutar el proyecto en XAMPP (Apache + MySQL)
5. Acceder desde el navegador:
http://localhost/matriculas_web_app

## 🌐 Demo del sistema
https://sistemamatriculas.rf.gd/dashboard.php

## 🧩 Estructura del sistema
El sistema está dividido en módulos:
- Alumnos
- Matrículas
- Pagos
- Reportes

## 🗄 Base de datos
El sistema utiliza una base de datos relacional en MySQL, con integridad referencial mediante claves foráneas entre las tablas principales:
- alumnos
- matrículas
- pagos
- grados

INSTRUCCIONES RÁPIDAS:
1. Coloca esta carpeta en htdocs (XAMPP) -> e.g. C:\xampp\htdocs\matriculas_web_app
2. Importa el archivo 'database/db_final.sql' en phpMyAdmin (creará la base y tablas)
3. Abre http://localhost/matriculas_web_app/dashboard.php
4. Usuario demo: admin / admin (cambiar en production)
5. Para cargar alumnos desde Excel: guarda CSV con columnas apellidos_nombres,dni,fecha_nacimiento,apoderado,correo,grado_id,seccion_id y usa Importar en phpMyAdmin o el formulario 'Alumnos' para agregarlos.
